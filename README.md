# English Wave — персональный тренажёр английского

Одностраничное веб-приложение для самостоятельной практики английского языка: словарь, чтение, аудирование, письмо, говорение (с распознаванием произношения), оригинальные кино-сцены и ежемесячный тест прогресса.

Реализовано по [PRD.md](./PRD.md).

## Как начать пользоваться

1. Открой сайт (ссылка — в описании репозитория / выдана после публикации).
2. При первом запуске откроется окно **Настройки** — выбери провайдера и вставь свой ключ:
   - **Anthropic — Claude**: ключ с [claude.com/settings/keys](https://claude.com/settings/keys);
   - **Z.ai — GLM-5.2**: ключ с [z.ai/model-api](https://z.ai/model-api).
3. Ключ сохраняется **только в твоём браузере** (`localStorage`) и используется для прямых запросов к API выбранного провайдера — ни на какой сторонний сервер, ни в репозиторий он не передаётся.
4. Выбери уровень (Начальный / Средний / Продвинутый) и начинай заниматься.

> ⚠️ Ключи API — это личные секреты. Никогда не храните их в коде репозитория (даже приватного и даже "спрятанными" под неочевидным именем файла) — публичные и приватные репозитории регулярно сканируются ботами на утечки, а обфускация имени файла не является защитой. Правильное место для личного ключа — только локальное хранилище браузера, как сделано в этом приложении.

## Технологии

- Чистый HTML/CSS/JS, без сборки и зависимостей — один файл `index.html`.
- Генерация контента — на выбор пользователя:
  - Anthropic Messages API (`fetch` напрямую из браузера, заголовок `anthropic-dangerous-direct-browser-access`);
  - Z.ai GLM-5.2, OpenAI-совместимый Chat Completions API (`https://api.z.ai/api/paas/v4/chat/completions`, прямой CORS-запрос из браузера).
- Озвучка — `SpeechSynthesis` (`en-US`).
- Распознавание речи для практики произношения — `SpeechRecognition` / `webkitSpeechRecognition` (лучше всего работает в Chrome), сравнение с эталонной фразой пословно по алгоритму LCS.
- Прогресс, стрик, настройки (включая оба API-ключа) и история тестов — `localStorage`, ничего не покидает устройство пользователя, кроме запросов к API выбранного провайдера.
- Авторское право: контент (тексты, диалоги, кино-сцены) генерируется полностью оригинально, без реальных цитат из фильмов/сериалов/песен.

## Локальный запуск

Файл статический — достаточно открыть `index.html` в браузере или поднять любой статический сервер, например:

```bash
python -m http.server 8420
```
