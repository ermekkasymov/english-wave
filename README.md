# English Wave — персональный тренажёр английского

Одностраничное веб-приложение для самостоятельной практики английского языка: словарь, чтение, аудирование, письмо, говорение (с распознаванием произношения), оригинальные кино-сцены и ежемесячный тест прогресса.

Реализовано по [PRD.md](./PRD.md).

## Как начать пользоваться

1. Открой сайт (ссылка — в описании репозитория / выдана после публикации).
2. При первом запуске откроется окно **Настройки** — вставь туда свой ключ Anthropic API (получить можно на [claude.com/settings/keys](https://claude.com/settings/keys)).
3. Ключ сохраняется **только в твоём браузере** (`localStorage`) и используется для прямых запросов к Anthropic API — на сторонний сервер он не передаётся.
4. Выбери уровень (Начальный / Средний / Продвинутый) и начинай заниматься.

## Технологии

- Чистый HTML/CSS/JS, без сборки и зависимостей — один файл `index.html`.
- Генерация контента — Anthropic Messages API (`fetch` напрямую из браузера, заголовок `anthropic-dangerous-direct-browser-access`).
- Озвучка — `SpeechSynthesis` (`en-US`).
- Распознавание речи для практики произношения — `SpeechRecognition` / `webkitSpeechRecognition` (лучше всего работает в Chrome), сравнение с эталонной фразой пословно по алгоритму LCS.
- Прогресс, стрик, настройки и история тестов — `localStorage`, ничего не покидает устройство пользователя, кроме запросов к Anthropic API.
- Авторское право: контент (тексты, диалоги, кино-сцены) генерируется полностью оригинально, без реальных цитат из фильмов/сериалов/песен.

## Локальный запуск

Файл статический — достаточно открыть `index.html` в браузере или поднять любой статический сервер, например:

```bash
python -m http.server 8420
```
